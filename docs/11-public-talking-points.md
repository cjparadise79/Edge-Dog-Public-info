# Public Talking Points

## Short Description

Edge Dog is a KARIOS Edge robotics prototype running on Raspberry Pi 5-class local compute. It uses EdgeLang, a deterministic programming language and runtime, to structure safe robot behavior at the edge while KARIOS Core supports higher-level reasoning in the cloud.

## Elevator Pitch

Edge Dog is not a chatbot on four legs. It is an edge-designed autonomy prototype that listens, reasons locally, checks safety, executes bounded robot behaviors, and exposes live video and telemetry through a desktop operator app.

## Partner Pitch

Edge Dog demonstrates a practical autonomy split:

- local edge runtime for safety, sensors, and action
- EdgeLang for deterministic mission execution
- cloud reasoning through KARIOS Core for deeper conversation and planning support
- desktop operator visibility for supervision and recording

## Demo Framing

When presenting Edge Dog, focus on:

- what is running locally
- how safety gates work
- how EdgeLang structures behavior
- how the desktop app exposes video and telemetry
- how KARIOS Core supports higher-level reasoning without taking direct motor control

## Avoid Overclaiming

Use "prototype," "early runtime," "demonstration," and "architecture proof" when describing current status.

Avoid implying:

- production certification
- finished field deployment
- unrestricted autonomous navigation
- cloud-controlled motor authority
- human-level general robotics intelligence

## Best One-Liner

Edge Dog is a small-scale field autonomy prototype: KARIOS Edge controls local safety and action, EdgeLang defines deterministic robot behavior, and KARIOS Core provides higher reasoning without taking motor authority away from the robot.
