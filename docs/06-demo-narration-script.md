# Demo Narration Script

This script is written for a short public-facing Edge Dog video of about one minute.

## 68-Second Version

Meet Edge Dog, a KARIOS Edge robotics prototype running locally on a Raspberry Pi 5.

This is intentionally minimal hardware: onboard sensors, local compute, legged motion control, video, audio, and a safety-first runtime built to prove autonomy at the edge.

This is not a chatbot on four legs. Edge Dog is an edge-designed AI system built around the EdgeLang programming language and runtime.

EdgeLang defines missions, checks, actions, fallbacks, and traceable behavior, so the robot can move from user intent to controlled execution instead of guessing blindly.

Edge can listen, respond, move, stop, scan, and begin making local decisions from sensor feedback. Motor authority stays on the robot. Obstacle checks, balance feedback, emergency stop, and safety refusal happen locally, where timing matters.

For deeper conversation and higher-level reasoning, Edge can hand off to KARIOS Core in the cloud. But KARIOS Core advises. Edge decides what is safe to execute.

The desktop operator app gives a technician live video, telemetry, remote control, audio interaction, and recording, so the system can be observed while it operates.

This is an early prototype, but the architecture is the point: local autonomy, explainable runtime behavior, cloud-assisted reasoning, and safety-first control on minimal edge hardware.
