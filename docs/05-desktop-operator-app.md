# Desktop Operator App

The Edge Dog desktop operator app is a technician-facing control and observation tool. It is designed to make the robot's state visible while the robot is operating.

## Purpose

The app supports the idea that autonomous systems should not be opaque. A human operator should be able to see what the robot sees, inspect runtime state, and supervise behavior.

## Public Feature Summary

The app is designed to support:

- live video streaming from the robot
- low-latency visual monitoring
- remote command input
- motion/status visibility
- telemetry inspection
- voice/audio interaction support
- recording of visual sessions
- operator supervision during demos

## Demo Role

In a live demo, the desktop app helps show that Edge Dog is not just moving blindly. It gives the operator and audience a view into:

- what the robot is doing
- whether the robot is connected
- what video stream is available
- whether motion is active
- whether the system is responding

## Why It Matters

For robotics partners, operator visibility is often as important as autonomy. The app demonstrates a practical control-room pattern:

- robot at the edge
- operator at a workstation
- live video and telemetry between them
- local safety on the robot
- remote supervision without hiding the machine state
